# 西兰花炒胡萝卜的做法
减脂必备

# 原料
西兰花
胡萝卜
盐
面粉
油
大蒜
生抽
鸡精
# 操作
1. 西兰花处理<br>

掰成小块，放入水中，加盐、面粉，浸泡10-20分钟，去除农药残留等

2. 胡萝卜处理<br>

胡萝卜洗干净，切片

3. 西兰花、胡萝卜焯水<br>

水开放入少量盐，放入西兰花、胡萝卜，大火焯水三分钟。捞出，放到凉水中

4. 大蒜切成末<br>

5. 翻炒<br>

- 锅热到油，蒜末炒出蒜香
- 倒入西兰花、胡萝卜，大火翻炒2-3分钟
- 倒入少许生抽、鸡精，盐由于处理时放了两次，可以添加少许，控制好量
